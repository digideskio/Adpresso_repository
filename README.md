Adpresso_repository
===========

- Adpresso Android sdk.


Requirement
=========

- Android version 4.0.3 ( API level 15 )
- Google Play Service

How to use
=========

- add your build.gradle file

```
repositories {
  maven {
    url 'https://valuecommerce.github.io/Adpresso_repository/'
  }
}
```

- add your build.gradle file

```
compile 'com.valuecommerce.android:android-sdk:2.0.0'
```
